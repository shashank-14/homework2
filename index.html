Implementing HTML Tags

This project includes various HTML tags. HTML is a markup language and makes use of various tags to format the content. These tags are enclosed within angle braces.

The HTML b tag specifies bold text

The HTML i tag specifies italic text.

Browsers generally render blockquote text as indented text. If your quoted text needs to display within a non-quoted paragraph, you should use the HTML q tag. Most browsers surround q text with quotation marks.
